Como Instalar o Widget NPS em Seu Site

Existem várias maneiras de adicionar o widget NPS ao seu site. Escolha a opção que melhor se adapta às suas necessidades.

Método 1: Instalação Automática

Adicione este script ao seu HTML para instalar automaticamente o widget:

<script 
    src="https://seu-dominio.com/install-nps-widget.js" 
    data-auto-init="true" 
    data-config='{"userId": "cliente123", "primaryColor": "#ea5f3d"}'
></script>

O widget será automaticamente inicializado com as configurações fornecidas.

Método 2: Instalação Manual

Adicione o script e inicialize o widget manualmente:

<script src="https://seu-dominio.com/install-nps-widget.js"></script>
<script>
    // Inicializar o widget quando desejar
    document.getElementById('mostrarFeedback').addEventListener('click', function() {
        initNPSWidget({
            userId: 'cliente123',
            primaryColor: '#ea5f3d',
            title: 'Como foi sua experiência?'
        });
    });
</script>

Método 3: Instalação em Elemento Específico

Você pode especificar onde o widget deve aparecer:

<div id="meu-widget-nps"></div>

<script src="https://seu-dominio.com/install-nps-widget.js"></script>
<script>
    initNPSWidget({
        targetElementId: 'meu-widget-nps',
        userId: 'cliente123'
    });
</script>

Opções de Configuração

O widget aceita as seguintes opções de configuração:

  • targetElementId: ID do elemento onde o widget será exibido
  • userId: Identificador do usuário
  • apiUrl: URL da API para enviar os dados
  • apiKey: Chave de API para autenticação
  • title: Texto da pergunta principal
  • feedbackLabel: Texto para o campo de feedback
  • submitButtonText: Texto do botão de envio
  • thankYouMessage: Mensagem de agradecimento
  • primaryColor: Cor principal do widget (código hex)
  • maxRating: Valor máximo da avaliação (padrão: 10)
  • darkMode: Ativar modo escuro (true/false)

Manipulação de Dados Personalizada

Você pode personalizar como os dados são processados:

initNPSWidget({
    onSubmit: function(data) {
        // Dados contêm: user_id, score, feedback, created_at
        console.log('Feedback recebido:', data);
        
        // Enviar para seu próprio sistema
        return fetch('https://seu-sistema.com/api/feedback', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify(data)
        });
    }
});

Nota: Substitua "https://seu-dominio.com/install-nps-widget.js" pelo caminho real do script em seu servidor.
